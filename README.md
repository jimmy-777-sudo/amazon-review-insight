# Amazon Review Insight

AI-powered Amazon review analysis tool for cross-border sellers.

Input an ASIN, get a pain-point analysis report extracted from 1-3 star reviews — instantly.

## Why this exists

Cross-border sellers spend hours manually reading competitor reviews to find product gaps. This tool automates that with AI.

## Quick Start

```bash
git clone https://github.com/jimmy-777-sudo/amazon-review-insight.git
cd amazon-review-insight
pip install -r requirements.txt
python app.py
```
